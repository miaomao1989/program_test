static void rte_jhash_2hashes ( const void *  key,
uint32_t  length,
uint32_t *  pc,
uint32_t *  pb 
)
inlinestatic

Same as rte_jhash, but takes two seeds and return two uint32_ts. pc and pb must be non-null, and *pc and *pb must both be initialized with seeds. If you pass in (*pb)=0, the output (*pc) will be the same as the return value from rte_jhash.

Parameters
keyKey to calculate hash of.
lengthLength of key in bytes.
pcIN: seed OUT: primary hash value.
pbIN: second seed OUT: secondary hash value.
